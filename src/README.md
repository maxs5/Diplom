# Finance App — Приложение для учёта финансов

Современное веб-приложение для управления личными финансами, созданное с использованием React и Tailwind CSS.

## Основные возможности

### Авторизация
- Регистрация нового пользователя
- Вход в существующий аккаунт
- Безопасное хранение данных в localStorage

### Главная страница (Dashboard)
- **Статистика** — общий баланс, доходы, расходы за период
- **Графики** — визуализация доходов/расходов по месяцам
- **Аналитика** — топ категорий расходов
- **Счета** — список всех счетов с балансами
- **Категории** — быстрый доступ к категориям

### Управление счетами
- Создание новых счетов (карты, наличные, вклады и т.д.)
- Редактирование существующих счетов
- Удаление счетов
- Автоматический подсчёт баланса
- Выбор типа счёта с иконкой

### Управление категориями
- Создание пользовательских категорий
- Разделение на доходы и расходы
- 40+ иконок на выбор
- Редактирование и удаление категорий

### Операции (доходы/расходы)
- **Создание операций** — добавление доходов и расходов
- **Редактирование** — изменение существующих операций
- **Удаление** — удаление операций с пересчётом баланса
- **Комментарии** — заметки к каждой операции
- **Автоматический пересчёт** — балансы обновляются мгновенно

### История операций
- Просмотр всех операций
- **Мощные фильтры:**
  - По типу (доход/расход)
  - По счёту
  - По категории
  - По диапазону дат
- **Пагинация** — удобный просмотр больших списков
- Сортировка по дате

### Настройки
- Изменение имени пользователя
- Изменение email
- Выбор валюты (₽, $, €)
- Информация об аккаунте

## Дизайн и UX

- **Адаптивный дизайн** — работает на всех устройствах
- **Тёмная/светлая тема** — автоматическое определение

## Как использовать

### Демо-аккаунт
Для быстрого тестирования используйте демо-аккаунт:
- **Email:** demo@test.com
- **Пароль:** 123456

Демо-аккаунт содержит примеры:
- 3 счета с балансами
- 8 категорий (доходы и расходы)
- 20+ операций за последние месяцы

### Создание нового аккаунта
1. Перейдите на страницу регистрации
2. Введите имя, email и пароль (минимум 6 символов)
3. Нажмите "Зарегистрироваться"
4. Автоматически создадутся дефолтные категории

### Работа с приложением

#### Создание счёта
1. Главная → "Добавить" в секции "Мои счета"
2. Введите название (например: "Основная карта")
3. Выберите тип счёта
4. Укажите начальный баланс
5. Сохраните

#### Создание категории
1. Главная → "Добавить" в секции "Категории"
2. Введите название
3. Выберите тип (доход/расход)
4. Выберите иконку из 40+ вариантов
5. Сохраните

#### Добавление операции
1. Боковое меню → "Новая операция"
2. Выберите тип (доход или расход)
3. Выберите счёт
4. Выберите категорию (зависит от типа)
5. Введите сумму
6. Выберите дату
7. Добавьте комментарий (опционально)
8. Сохраните

Баланс счёта обновится автоматически!

#### Просмотр истории
1. Боковое меню → "История операций"
2. Используйте фильтры для поиска:
   - Тип операции
   - Счёт
   - Категория
   - Диапазон дат
3. Нажмите на операцию для редактирования
4. Или удалите её кнопкой 🗑️

## Особенности реализации

### Архитектура
- **Context API** — глобальное управление состоянием
- **Custom Hooks** — переиспользуемая логика
- **Feature-Sliced Design** — организация кода
- **Component Composition** — композиция компонентов

### Хранение данных
Все данные хранятся в `localStorage` браузера:
- `financeApp_users` — список пользователей
- `financeApp_user` — текущая сессия
- `financeApp_accounts` — счета
- `financeApp_categories` — категории
- `financeApp_operations` — операции

### Автоматический пересчёт балансов
При создании/редактировании/удалении операции:
1. Система отменяет старое изменение баланса (если есть)
2. Применяет новое изменение
3. Сохраняет обновлённый баланс счёта

### Валидация форм
Все формы содержат валидацию:
- Обязательные поля отмечены *
- Проверка email формата
- Минимальная длина пароля
- Положительные суммы
- Корректные даты

## Адаптивность

Приложение полностью адаптивно:
- **Desktop (1024px+)** — полный функционал, боковое меню всегда видимо
- **Tablet (768-1023px)** — боковое меню скрывается, открывается кнопкой
- **Mobile (до 767px)** — оптимизированный интерфейс, touch-friendly

## Кастомизация

### Цветовая схема
Все цвета определены в CSS-переменных (`styles/globals.css`):
```css
--color-primary: #6366f1;      /* Основной цвет */
--color-success: #10b981;      /* Доходы */
--color-danger: #ef4444;       /* Расходы */
```

### Добавление новых иконок
Редактируйте `data/constants.js`:
```javascript
export const CATEGORY_ICONS = [
  { id: 'my-icon', emoji: '🎉', label: 'Моя иконка' },
  // ...
];
```

### Добавление типов счетов
Редактируйте `data/constants.js`:
```javascript
export const ACCOUNT_TYPES = [
  { id: 'my-type', label: 'Мой тип', icon: '💎' },
  // ...
];
```
